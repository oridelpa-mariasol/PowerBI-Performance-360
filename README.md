Performance 360: Supply Chain & Sales Analysis

 📊 Descripción : Este proyecto presenta un análisis integral de una distribuidora de consumo masivo en Uruguay.
 El objetivo es transformar datos operativos en insights estratégicos, integrando métricas de ventas con eficiencia logística y rentabilidad.

 🛠️ Tecnologías y Herramientas :  Power BI - Modelado de datos y diseño de dashboards.
