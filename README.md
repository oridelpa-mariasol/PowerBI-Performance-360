Executive Sales & CRM Dashboard - Case Study

1 - Objetivo del Proyecto: Este proyecto presenta una estructura de datos diseñado para monitorear el desempeño comercial y la eficiencia de la gestión de clientes (CRM). El objetivo principal es transformar datos transaccionales y operativos en insights estratégicos para optimizar la toma de decisiones.

2. Aspectos Técnicos y Preparación de Datos
Ingesta de Datos: Se trabajó con una base de datos estructurada y limpia, lo que permitió una implementación ágil.La misma fue generada a traves de Gemini, donde se genero un prompt indicando la estructura y tablas de la base. 

Modelado de Datos :  Vinculación eficiente de tablas de Ventas, Seguimientos y Clientes.

Creación de una Tabla de Calendario personalizada para unificar la inteligencia de tiempo y permitir comparaciones interanuales.

DAX & KPIs: Se centralizaron todas las métricas en una tabla técnica dedicada. Entre los cálculos principales se encuentran:

Tasa de Gestión de Ventas: Métrica clave que cruza los seguimientos del CRM con las ventas efectivas para medir la conversión.

Volumen de Seguimiento: Análisis de intensidad de contacto por cliente.

Segmentación de Contactabilidad: Porcentaje de clientes contactados vs. no contactados.

3. Visualización y Estructura (3 Dashboards)
Performance 360: Vista ejecutiva con la evolución de ventas 2025 vs. 2026 y KPIs principales.

CRM & Productividad: Detalle del esfuerzo del equipo de ventas, canales de contacto y efectividad de gestión.

Análisis de Productos y Proveedores: Enfoque en el inventario, identificando los rubros y marcas líderes en volumen de ventas.

4. Business Insights & Hallazgos Estratégicos
Tras el análisis del dashboard, se identificaron los siguientes puntos críticos:

Análisis Estacional: Se detectó una baja en las ventas durante el primer trimestre. Se propone el diseño de promociones para los meses de invierno para mitigar la estacionalidad.

Eficiencia del Canal: El canal de WhatsApp es el más utilizado. Se recomienda profundizar en la Tasa de Gestión por Canal para determinar si es el más efectivo en conversión o solo en volumen.

Gestión de Ventas: Se identificó una disparidad en la tasa de cierre entre vendedores.

Plan de Acción Propuesto:

Incentivos: Crear un porcentaje extra de comisión para vendedores que alcancen metas de conversión específicas.

Meta de Contactabilidad: Establecer un objetivo del 100% de clientes contactados por vendedor para garantizar que no haya "fugas" en la cartera.

Benchmarking: Realizar un análisis de crecimiento (Growth Rate) mes a mes para monitorear la recuperación de la tendencia.
